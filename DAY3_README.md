# Day 3 — Сравнение способов рассуждения через API

## Что делает скрипт
`day3_reasoning_compare.py` решает **одну и ту же задачу** 4 способами:

1. Прямой ответ (без доп. инструкций)
2. С инструкцией: `решай пошагово`
3. Сначала модель генерирует промпт для решения, потом решает им
4. Multi-expert: аналитик, инженер, критик

После этого формируется сравнение:
- отличаются ли ответы
- сколько ответов корректны
- какой подход выглядит лучшим

Результат сохраняется в `day3_results.json`.

---

## Запуск

```bash
cd /root/.openclaw/workspace/AI-Challenge
export OPENAI_API_KEY="<YOUR_KEY>"
# Опционально:
# export OPENAI_MODEL="gpt-4o-mini"
# export OPENAI_BASE_URL="https://api.openai.com/v1"

python3 day3_reasoning_compare.py
```
