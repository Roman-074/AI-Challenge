# Day 4 — Температура (0 vs 0.7 vs 1.2)

## Что сделано
Скрипт `day4_temperature_compare.py` запускает **один и тот же запрос** с тремя настройками:
- `temperature = 0`
- `temperature = 0.7`
- `temperature = 1.2`

После запуска формируется файл `day4_temperature_results.json` с:
- примерами ответов,
- метриками (точность, длина, лексическое разнообразие, креативность),
- сравнением разнообразия относительно `temperature=0`,
- выводами по тому, для каких задач лучше каждая температура.

## Запуск
```bash
cd /root/.openclaw/workspace/AI-Challenge
export OPENAI_API_KEY="<YOUR_KEY>"
# опционально:
# export OPENAI_MODEL="gpt-4o-mini"
# export OPENAI_BASE_URL="https://api.openai.com/v1"

python3 day4_temperature_compare.py
```

## Что проверяем в результатах
1. **Точность** — присутствует ли ожидаемый факт (для этой задачи: ответ `4`).
2. **Креативность** — оценка по длине + лексическому разнообразию.
3. **Разнообразие** — отличие от ответа на `temperature=0`.

## Практические выводы
- `temperature=0` — лучший выбор для точных и повторяемых задач (вычисления, тех.ответы, регламенты).
- `temperature=0.7` — универсальный баланс для большинства продуктовых и бизнес-задач.
- `temperature=1.2` — для идей, вариативности, креативных формулировок и брейншторминга.
